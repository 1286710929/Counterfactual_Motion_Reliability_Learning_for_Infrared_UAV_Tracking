# Checkpoint

The best CMRTrack checkpoint used for the reported Anti-UAV410 result is:

```text
OSTrack_ep0030.pth.tar
```

It is not committed to this GitHub repository because the file is about 1.1 GB, which exceeds the normal GitHub file-size limit.

Download it from the GitHub Release page:

```text
https://github.com/1286710929/Counterfactual_Motion_Reliability_Learning_for_Infrared_UAV_Tracking/releases/download/v1.0.0/OSTrack_ep0030.pth.tar
```

Then place the checkpoint at:

```text
checkpoints/OSTrack_ep0030.pth.tar
```

SHA256:

```text
f3236be0aa911550a100c7f7940885148b4bc498d417f661e7ebb3b70d49d737
```

Verify the file with:

```bash
sha256sum checkpoints/OSTrack_ep0030.pth.tar
```
